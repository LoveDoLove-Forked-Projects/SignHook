## SignHook(签名助手) Xp模块

## 使用说明

* 注册一个账号，填写邮箱即可注册，注册成功后会收到一封邮件，里面包含密码，请妥善保管。

* 目前只支持某Q和某V,其他后续会陆续支持,有需要适配的可以联系或者自行fork修改

* 作用域的应用启动后会要求登录一次，之后在主页面和分享页面会请求服务器下发已选中的包的信息
